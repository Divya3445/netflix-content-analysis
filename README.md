# netflix-content-analysis
Netflix Content Analysis project using Python and Power BI to explore content trends, genres, and country-wise distribution through an interactive dashboard.

# Netflix Content Analysis 📊

## Project Overview

This project analyzes the Netflix Titles dataset to explore patterns and trends in movies and TV shows available on the Netflix platform. The objective of this analysis is to understand how Netflix content is distributed across different types, countries, release years, and genres.

The project demonstrates practical data analysis skills using **Python for exploratory data analysis (EDA)** and **Power BI for building an interactive dashboard**. The insights obtained from this analysis help identify content trends and key characteristics of Netflix's content library.

---

## Objectives

The main objectives of this project are:

* Analyze the distribution of **Movies vs TV Shows** on Netflix
* Identify **content trends over the years**
* Determine the **top countries producing Netflix content**
* Identify **popular genres available on Netflix**
* Create an **interactive dashboard** to visualize insights

---

## Dataset

The dataset used in this project is the **Netflix Titles Dataset**, which contains information about movies and TV shows available on Netflix.

Key columns include:

* `show_id` – Unique identifier for each title
* `title` – Name of the movie or TV show
* `type` – Movie or TV Show
* `director` – Director of the content
* `country` – Country where the content was produced
* `release_year` – Year of release
* `rating` – Content rating
* `duration` – Length of the content
* `listed_in` – Genre category

---

## Exploratory Data Analysis (Python)

Using Python, the dataset was cleaned and explored to understand the structure and identify patterns.

Key analysis performed:

* Distribution of movies and TV shows
* Content released over the years
* Top countries producing Netflix content
* Most common genres on Netflix

Python libraries such as **Pandas, Matplotlib, and Seaborn** were used to perform data analysis and visualization.

## Power BI Dashboard

An interactive Power BI dashboard was created to visualize the insights from the dataset.

### Dashboard Components

* **KPI Cards**

  * Total Titles
  * Total Movies
  * Total TV Shows

* **Pie Chart**

  * Movies vs TV Shows distribution

* **Line Chart**

  * Netflix content released over years

* **Bar Charts**

  * Top countries producing Netflix content
  * Most common Netflix genres

* **Interactive Filters**

  * Release Year
  * Country
  * Rating

---

## Key Insights

* Movies represent the majority of Netflix content compared to TV shows.
* Netflix content increased significantly after **2015**, indicating rapid platform expansion.
* The **United States** produces the highest number of Netflix titles.
* Genres such as **Documentaries, Stand-Up Comedy, and International Dramas** appear frequently in the dataset.
